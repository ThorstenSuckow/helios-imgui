## helios::imgui::widgets

Collection of debug and developer widgets for ImGui overlays. Widgets visualize runtime state (FPS, input, logging, profiling, camera control) and provide interactive configuration interfaces.


---
<details>
<summary>Doxygen</summary><p>
@namespace helios::imgui::widgets
@brief Debug and developer widgets for ImGui overlays.
@details This namespace contains pre-built widgets for common development/debugging tasks such as FPS monitoring, gamepad input visualization, log output, camera control, and profiling. All widgets inherit from `ImGuiWidget` and are intended for use with `ImGuiOverlay`.
</p></details>
